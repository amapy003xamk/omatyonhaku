Sovellusohjelmointi-kurssin lopputyö: Oma työnhaku -sovellus

Muistiosovellus omien työnhakuun liittyvien tietojen kirjaamiseen
ja ylläpitämiseen. Sovelluksella voi lisätä mielenkiintoisia
työpaikkoja muistiin, kirjata niihin liittyviä muistiinpanoja ja 
poistaa tietoja. Sovellukseen tulee rekisteröityä ja kirjautua.
Tietokantana sovellukseen tallennettavilla tiedoilla toimii MySQL.

sql -kansiosta löytyy SQL-dumppi tietokannan luomiseen.

Harjoituksessa tuli käyttää Node.js:ää ja kurssin aikana
opeteltuja tekniikoita.

Ohjelmassa on käytetty seuraavia npm-paketteja ja tekniikoita:
 - dot env
 - ejs
 - express
 - express-session
 - mysql

Muut laajennukset:
 - CKEditor 4
 - Bootstrap 4.4.1 + popper.js
 - jQuery 3.4.1
